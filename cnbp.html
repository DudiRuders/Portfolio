CNBP — czyszczenie i normalizacja bazy produktów

Pipeline do ujednolicania brudnych danych produktowych (nazwy + gramatury + skróty + literówki) z wielu źródeł, z raportem wyjątków i spójnym outputem do dalszych procesów.

Problem

  • Niespójne nazwy produktów, literówki, skróty.
  • Gramatura i jednostki w różnych formatach.
  • Dane z wielu działów → brak standardu.

Rozwiązanie

  • Reguły czyszczenia tekstu + normalizacja.
  • Ekstrakcja gramatur (regex).
  • Słowniki zamian + lista wyjątków do ręcznej walidacji.
  • Output: CSV i/lub PostgreSQL.

Stack

Python Pandas Regex CSV/XLSX PostgreSQL (opc.)
